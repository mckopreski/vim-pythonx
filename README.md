# Python libraries for smarter Vim coding

## Installation

Just add plugin using [Plug](https://github.com/junegunn/vim-plug):

```viml
Plug 'reconquest/vim-pythonx'
```

Import required py-modules in snippets files (e.g. in `go.snippets`):

```
global !p
import px.langs.go
endglobal
```
